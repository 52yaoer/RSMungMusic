## MungMusicPlayer
Android端音乐播放器，目前有两个页面。  
版本1.3中删除了登录功能,因为我觉得好像没什么用  
#### 基本功能:  
1.与服务器使用原生socket通信验证账号密码(服务器可放在公网),默认账号密码 `rs`  `123456789`  
2.显示本地歌曲  
3.播放本地歌曲(可后台播放)  
4.记录播放历史  
5.搜索本地歌曲  
6.定时停止播放  
7.播放模式选择  
8.进度条显示歌曲播放进度
